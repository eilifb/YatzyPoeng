Sjekker om det er et antall like terningverdier i et kast og summerer de terningverdiene.

Hvor mange like funksjonen ser etter tar den ikke hensyn til, utover det spesifisert i antallLike. Så man kan bryte vanlige Yatzyregler her hvis man setter antallLike til noe annet en 3 eller 4.

Parameters

Returns
Summen av terningene som har lik terningverdi, eller 0 om det ikke er nok antall like terningverdier.

Parameters
terningerInput5 terningverdier separert med komma

Et Tuple<string,int>(), hvor strengen er navnet på kategorien, og tallet er poengsummen. I tilfelle hvor flere kategorier gir samme sum så prioriteres den som er listet øverst i oppgaven, fult hus over sjanse osv.
